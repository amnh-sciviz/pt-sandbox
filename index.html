Periodic Table Sandbox

An internal website showing prototypes and works-in-progress related to the Periodic Table Interactive Media Installation in the Hall of Gems and Minerals. You can view source code here and documentation here.

Interaction tests

  1. Physics interaction - Demonstrate physics-based interaction of multiple objects
  2. Make or break - Combine elements into minerals, break apart minerals into elements
  3. Basic molecule interaction - Demonstration of simply combining atoms into molecules, highlighting the electrons
  4. Periodic Table - Interact with the periodic table

Utilities

  1. TUIO/Multi-touch test - A basic utility for testing/debugging multi-touch functionality on a TUIO device
  2. Multi-touch test - A basic utility for testing/debugging multi-touch functionality on a touch-enabled device
